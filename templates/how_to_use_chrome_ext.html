How To Use Chrome Extension

Options page is available from "Manage Extensions" page. Open your "Manage Extensions" page (this depends on your Chrome version), find a SyncTab plugin.


SyncTab Extension

And click on an Options link. You'll see a page like:


Options Page

View Currently Used Account

Open SyncTab Extension Options page. If you're registered now, you'll will see an Account item with a currently registered user email.

Logout

Open SyncTab Extension Options page. If you're registered now, you'll will see an Account item with a Logout button. Click on Logout button.

Add Shared Links to Bookmarks

Open SyncTab Extension Options page. Select Yes for Add Tabs to Bookmarks option.

Enable or Disable Notifications

Open SyncTab Extension Options page. Select Yes or No for Show Notifications option.
