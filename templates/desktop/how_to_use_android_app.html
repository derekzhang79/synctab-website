How To Use Android Application

Send a Link

Lets try to send a link from Android browser. Start a browser application on your phone and open some page in it, e.g. http://google.com. Click on Menu and search for "Share page" option. After you select it, you will see a "Share via" dialog with a list of applications you want to share with. Find and select a SyncTab option here.

Send link via SyncTab

After you select a SyncTab option, you will see a new screen from SyncTab application. Here you will see a dialog box called "Send To..." with a few tags. You can send a link with specified tag only. Such you limit what devices and browsers can receive and open link. For example, you can send a link to Home browser, or to Android tablet.

Send To

Then you will see a message that a tab is syncing. Wait for a few seconds. If you have an Internet connection up, you will see a next message:

Shared Link

Now the link is shared. And as soon as you'll have a connection to the internet, it will be available in your desktop browser.

View List of Sent Links

Start SyncTab application. Login if need. You should be on Home page now. Home page contains a list of already shared links. When you connected to the internet, you may load the entire list of shared links.


List of Shared Links

Context Menu

To open a context menu for any shared link, just make a long click on it. After a few seconds application will open a context menu like this:

Context Menu

Re-send a Link

You may re-send a link again. Re-send used to send the link with specified tag again, so it can be opened again in your browser. Make a long click on the link you want to re-send, and press the Re-send option.

Remove a Link

You may remove a link. This link will also be removed from history. Just select a Remove option in context menu for shared link.

Copy Link into Clipboard

You may copy a link into clipboard buffer to past it somewhere else. Just select a Copy Link option in context menu for shared link.
