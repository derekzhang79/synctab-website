How To Use Chrome Extension

Options page is available from "Manage Extensions" page. Open your "Manage Extensions" page (this depends on your Chrome version), find a SyncTab plugin.


SyncTab Extension

And click on an Options link. You'll see a page like:


Options Page

View Currently Used Account

Open SyncTab Extension Options page. If you're registered now, you'll will see an Account item with a currently registered user email.

Change Browser Tag

Tag used to filter the links to open in the browser. For example, you can select current browser tag to be Home, so any links send with tag Hope will be opened in this browser. Such, you may have different Chrome browsers and have different tags for them. For example, one browser can be installed in your work computer and another in your home computer. Having them different current tags allows you to send links between them. Funny page can be send to the Home browser, while some professional article can be send to the Work browser. And you don't need to have them both online to send links. You may send a link from Home computer, then turn it off, come to work, turn on your work computer, open browser and in a minute browser will receive links.

Default Chrome browser tag is Chrome. But you can change it to be Home or Work.

Logout

Open SyncTab Extension Options page. If you're registered now, you'll will see an Account item with a Logout button. Click on Logout button.

Add Shared Links to Bookmarks

Open SyncTab Extension Options page. Select Yes for Add Tabs to Bookmarks option.

Enable or Disable Notifications

Open SyncTab Extension Options page. Select Yes or No for Show Notifications option.

Send a Link

You can send a link from Chrome browser to another Chrome browser or Android device. There is an SyncTab icon in the toolbar on the right side of the address bar.

SyncTab toolbar icon
Click on this icon to see the list of tags. Click on any tag to share a current tab page with specified tag.
If you right-click on any link on the page, you'll see a SyncTab item in the content menu. Use it to share a link with specified tag without even opening it!
