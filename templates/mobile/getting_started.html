Getting Started

{% endblock %} {% block content %}

This page is to help you to start using SyncTab to share a links from Android phone to your desktop browser. First you will need to install Android application and then install Chrome extension. You will need to register an account if you don't have yet a SyncTab account. This account should be used to login into SyncTab Android application and Chrome extension.

Install Android Application

Download and install Android application on Download page.

After install you will find a new application on your Android phone called SyncTab. SyncTab icon looks like on image:
You'll see a Login screen when you open SyncTab application for first time. If you're already registered, then enter your Email, Password and click on Login button.
Login Screen
Otherwise click on the Register button and use Registration screen to create a new account. Enter your real email and a secure password. SyncTab doesn't restrict you on password size or complexity. It's on your own!
Registration Screen
After successful login/registration, you will be redirected to the Home screen with empty list with your shared tabs. Of course, this will be an empty list on the begin.
Empty List
List with Shared Links
Now you may close SyncTab application and continue to install Chrome Extension.

Install Chrome Extension

Download and install Chrome Extension from Download page.

After you install Chrome extension, a new tab will be opened in your browser. This tab contains a page with login form. User your SyncTab account email and password (as you have already registered when installed Android application, use the same email and password) to login. If you don't have a SyncTab account yet, you may register it by clicking on Register Now link.
Login Form
After success login, you may close this page.

Now your browser will be opening a shared links as a new tabs!

You can read more about how to use Chrome extension.

Share a Link

Here I describe how to share a link from Android phone to your desktop browser, like Chrome. Android platform allows you to send a content to different application. This content can be a link, a text or image. SyncTab supports only processing a links. This means, you can use numerous Android applications (like Browser, Google Reader and so on) to send a link to SyncTab application. SyncTab application will do it best to open this link in your desktop browser.

There are a few ways to find such Send to action. It may be available to you from Menu by "Share", "Share via", "Share page", "Send" or "Send To" labels. You may also find it by it's standard icon, that looks like this:

Lets try to share a link from Android browser. Start a browser application on your phone and open some page with it, e.g. http://google.com. Click on Menu and search for "Share page" option. After you select it, you will see a "Share via" dialog with a list of applications you want to share with. Find and select a SyncTab option here.

Share via SyncTab

After you select a SyncTab option, you will see a new screen from SyncTab application. Here you will see a message that a tab is syncing. Wait for a few seconds. If you have an Internet connection up, you will see a next message:

Shared Link

And now, less than in a minute, your desktop browser will open a new tab with the shared link!
